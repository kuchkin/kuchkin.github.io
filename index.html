mumax3 + gneb

Note that this is an extension and is not available in the original mumax3 code!

mumax3+gneb is a GPU-accelerated GNEB method extension for Mumax3 program developed and maintained by at the Vlad Kuchkin at the University of Luxembourg.

Licence

mumax3+gneb is open-source software. You are free to modify and distribute the source code under the GPLv3 licence.

Features

  • Minimum energy path calculations
  • Eigenvalues calculation of the Hessian
  • Magnetostatic field
  • Heisenberg exchange
  • Dzyaloshinskii-Moriya interaction
  • Uniaxial magnetocrystalline anisotropy
  • Voronoi tessellation
  • Space dependent material parameters
  • Optional 1D, 2D or 3D periodic boundary conditions

web GUI

mumax3 includes a browser-based user interface that lets you follow a running simulation or modify it on-the-fly, be it on your local machine or remotely.

simple scripting

mumax3 provides simple yet powerful input scripting.
